# shmpipeline

`shmpipeline` builds local process-based compute pipelines on top of named shared-memory streams from `pyshmem`.

The documentation site is organized around four user paths:

- getting a pipeline running quickly
- understanding the CLI, runtime model, and GUI
- studying complete worked examples
- looking up the Python API and built-in kernel catalog
